UI Design and React Libraries

Fast development was the key requirement in order to succeed in the crowded marketplace, which drove the decision-making for UI design. We decided to utilize pre-existing React.js UI libraries as much as possible.

ag-Grid

We quickly decided to use ag-Grid which is one of the most robust grid libraries with JavaScript with a lot of customization possibilities. Since the grid is the key element in the platform, the rest of the UI needed to match with the look and feel of this library.

Blueprint JS

After researching UI libraries for React, we decided to go with Blueprint JS as it is suitable for building complex data-dense interfaces for desktop applications. Also, the minimalist look and feel match the rest of the UI.
