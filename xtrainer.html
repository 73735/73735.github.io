Elliptical Fitness Cross-trainer

Observation and Ethnographic Studies

Problem

An elliptical trainer or cross-trainer (also called an X-trainer) is a stationary exercise machine used to simulate stair climbing, walking, or running without causing excessive pressure to the joints, and can be used for both strength and cardio. It is gaining popularity in recent years. However, various levels of misuse of the machine have been reported. I conducted ethnographic studies to understand the problem.

Ground Tour

A ground tour interview was conducted with a male user in his 30's doing high-intensity interval training.

  1. Driving to the gym after work.
  2. Change.
  3. Enter the personal information (body weight).
  4. Adjust resistance (10).
  5. Start with fast pace using the handles (1 min).
  6. Cool down (30 sec).
  7. Speed up again.
  8. Repeat until going gets tough.
  9. Adjust resistance or pace to make it easy.
  10. Continue until going gets tough.
  11. Cool down.
  12. Wipe the handles before leaving.
  13. Shower and change.
  14. Going home driving.

Theme

People find it easy to use cross-trainers, however, they seems to lack the knowledge of what can be achieved using a cross-trainer and how to use one properly. The interface on the machine is complicated that people do not bother entering personal information such as weight which is important in tracking the progress of the exercise. People also do not add enough resistance ending in unnecessary long usage of the machine.

Insights

The lack of understanding of the machine has been the main theme for insights.

What Works

People seem to love using the machine.

Insight : Cross-trainers are less intimidating compared to treadmills and other machines at the gym.
User Need : I am not fit + I have trouble using a treadmill + I need something easy to use.

What Doesn't Works

People do not understand how to use cross-trainers correctly leading to inefficient use of the machine.

Insight : Cross-trainers are supposed to be intense, but it is hard to tell if someone is using the machine correctly.
User Need : I am new to the machine + I have trouble understanding what it does + I need a way to make sure if I am using the machine right.

Work-around

People tend to use cross-trainers for longer period of time in order to compensate for the ineffectiveness caused by improper use.

User Point of View

Problem : Need to lose weight using a cross-trainer.
Insight : Cross-trainers are easy to use, but hard to use efficiently.

An overweight person needs to lose weight. She is unfit to run on a treadmill, but she can pedal on a cross- trainer. She feels accomplished after her exercise, but not sure how something relatively easy can be an alternative for running on a treadmill.

Next Step

Cross-trainers are not intimidating and easy to use. It is a great starting point for those who otherwise do not come to the gym. Making it even simpler will bring more people to the gym. From my observations and interviews, it was revealed that most users are not using cross-trainers correctly. Some people use the machine without entering their weights or setting resistance leading to longer and inefficient use of the machine. The current interface has too many buttons with no instructions that users tend not to interact with it. As a solution, I created a prototype of a conversational interface with resistive touchscreen, which instruct users how to use the machine with wizard-style linier flow. The interface also acts as a personal trainer making sure that the user is training efficiently from start to finish. By using this prototype, I am trying to learn what steps are needed to teach user how to use the machine without overwhelming them.
