Teach Pendant

Membrane Design.

Casing

A teach pendant (CE Class B and FCC certified) was designed in order to operate Mujin’s Pick Worker®. The exterior was designed using only pre-existing components while board, cables, driver, membrane and emboss were custom-made.

Membrane

Three prototypes were made for membrane in order to select the best design. Usability testing was conducted using six participants who are familiar with the software. Each participants were asked to perform a simple task for each categories (general navigation, modeling, parameter editing and layout editing). All three designs were tested within subjects and evaluated quantitatively by participants and qualitatively by observers.

Prototype A

Prototype A is based on console game controllers and features arrow keys that are independent from selection keys. All buttons are made using Φ10mm circles.

The design is mainly intended for versatile use but also suitable for navigating through broad and shallow menu system.

Prototype B

The design in based on TV remote where the ‘ok’ key was surrounded by arrow keys. Design was executed using Φ8mm circles due to board design constraints.

It was design to perform well when navigating through narrow and deep menu system.

Prototype C

Prototype C is based on keyboards and features ‘up’ and ‘down’ keys placed closely next to each other whereas ‘right’ and ‘left’ buttons are separated.

The design is best suited for scrolling through vertical list items.

The participants preferred the prototype B for modeling and the prototype C for parameter editing. The prototype C was the most polarizing design as some participants rated at the top while others rated on the bottom. The observers noticed that the prototype C caused most confusions leading to numerous miss-clicks while the prototype A was least confusing.

Emboss

Corporate color orange was used on buttons and contours. Rectangular backgrounds were added on the function keys in order to make their connections with software keys clearer.
